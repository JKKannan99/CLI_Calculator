#CLI Calculator Toolkit

###This project is a simple Command Line Interface (CLI) calculator that performs basic arithmetic operations using Python.
###Users can run commands such as add, sub, or exit directly from the terminal.

##Features

1.Add,Subtract,multiply and divide two numbers

2.Exit the calculator

3.Easy to use from the command line

##How to Use

###Save the provided Python code in a file named calc.py

###pen Terminal or Command Prompt and navigate to the folder where calc.py is saved.

###Run Commands Addition like python calc.py add 12 10
